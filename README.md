# PersonalBudgetingExpenseTracker (NET48)

This version targets **.NET Framework 4.8** so WinForms Charting works:
- System.Windows.Forms.DataVisualization.Charting (Chart / Series / ChartArea)
- SQLite database via NuGet: System.Data.SQLite.Core

## Default login
- admin / admin123

## Run
Open the .sln in Visual Studio 2022+ and run.
Database file `budget.db` is created automatically in the output folder.
